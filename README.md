# 常識改変の街 — TSF Reality CYOA

> **常識改変とTSFの融合** — 住人は異変に気づくこともなく、変えられた姿で生活を送る。知っているのは「地域外」の私たちだけだ。

---

## 📖 概要

**常識改変の街** 是一款基于 React 的 CYOA（Choose Your Own Adventure）互动小说游戏。玩家扮演从「地域外」进入扭曲城镇的调查者，是唯一能察觉到异常的人。

### 核心概念

- 🌍 **常識改変** — 整个城镇的现实已被改写，但住民毫无知觉
- 👁️ **双重认知** — 切换「住民视角/真相视角」发现世界的扭曲
- 🧬 **TSF 身体变化** — 玩家自身也逐渐受到领域侵蚀
- 🎲 **随机身份反转** — 性别反转、亲子互换、师生颠倒、上下级逆转
- 👻 **魔物娘化** — 部分居民已转化为魔物娘，自己却觉得"本来就长这样"
- 🔐 **解谜小游戏** — 拼图、密码、记忆配对、找矛盾等谜题

### 游戏特色

- **Persona式天数系统** — 每天4时段（午前/午后/傍晚/夜间），有限行动次数，14天周目上限
- **12种BAD END** — 侵蚀率≥90%触发，不同魔物娘对应不同BAD END
- **7种正式结局** — 时间洞穴结构根据全部选择动态生成结局文本
- **6种CYOA分支结构** — 需要确认的选择 / 交换路线 / 计分选择 / 可重入节点 / 分院帽 / 时间洞穴
- **NPC好感度系统** — 5级好感度，多阶段对话，NPC有自己的日程
- **背包与线索系统** — 完整装备栏、线索关联推理、TSF触发道具
- **智能手机界面** — 相机/地图/笔记/图鉴/相册/信息等内置APP
- **多周目继承** — 发现记录、BAD END图鉴跨周目保留

## 🗺️ 区域结构

```
[外部世界] → [边界街区] → [中央商业区] → [扭曲核心]
   正常        10~30%侵蚀     30~60%侵蚀     60~100%侵蚀
```

- **边界街区**: 住宅区·便利店·旧书店·公园·地铁站
- **中央商业区**: 学校·医院·商店街·酒吧·图书馆·裁缝店·咖啡厅·游泳馆·花店
- **扭曲核心**: 市政大厅·神社·教堂·深渊回廊

## 🚀 本地运行

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/tsf-reality-cyoa.git
cd tsf-reality-cyoa

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 打包为单HTML文件
bash scripts/bundle-artifact.sh
```

## 📜 结局一览

### 正式结局
| 编号 | 名称 | 条件 |
|------|------|------|
| 真#1 | 現実解放者 | 认知度≥70 + 发现真相核心 |
| 真#2 | 天使的裁决 | 完成天使路线 + 侵蚀<30% |
| 真#3 | 龙之契约 | 找到龙女 + 做出正确选择 |
| N#1 | 漂流邊界 | 认知度40~60 |
| N#2 | 真相記錄者 | 发现≥25 + 认知度>50 |
| N#3 | 半人半魔 | 侵蚀率40~60 + 认知度30~50 |
| H#1 | 混沌居民 | 完成全部BAD END后解锁 |

### BAD END（12种）
每种对应不同的侵蚀触发条件和魔物娘路线。
